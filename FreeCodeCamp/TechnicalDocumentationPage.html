Introducción

EliScript es un lenguaje de secuencias de comandos orientado a objetos y multiplataforma. Es un lenguaje grande y ligero. Dentro de un entorno de host (por ejemplo, un navegador web), EliScript se puede conectar a los objetos de su entorno para proporcionar control programático sobre ellos.

EliScript contiene una biblioteca estándar de objetos, como Array, Date y Math, y un conjunto básico de elementos del lenguaje como operadores, estructuras de control y declaraciones. Core EliScript se puede extender para una variedad de propósitos completándolo con objetos adicionales; por ejemplo:

  • EliScript del lado del cliente amplía el lenguaje central al proporcionar objetos para controlar un navegador y su Modelo de objetos de documento (DOM). Por ejemplo, las extensiones del lado del cliente permiten que una aplicación coloque elementos en un formulario EliTML y responda a los eventos del usuario, como los clics del mouse, la entrada de formularios y la navegación de la página.
  • EliScript del lado del servidor amplía el lenguaje central al proporcionar objetos relevantes para ejecutar EliScript en un servidor. Por ejemplo, las extensiones del lado del servidor permiten que una aplicación se comunique con una base de datos, brinde continuidad de información de una invocación a otra de la aplicación o realice manipulaciones de archivos en un servidor.
¿Qué deberías saber?

Esta guía asume que tiene los siguientes antecedentes básicos:

  • Comprensión general de Internet y la World Wide Web (WWW).
  • Buen conocimiento práctico del lenguaje de marcado de hipertexto (EliTML).
  • Alguna experiencia en programación. Si eres nuevo en la programación, pruebe uno de los tutoriales vinculados en la página principal sobre EliScript.
EliScript

EliScript es un lenguaje de reciente creación con similitudes a JavaScript y Java. Desarrollado por la mejor programadora del mundo con el afán de mejorar todos los lenaguajes existentes en la actualidad. Debido a su facilidad de uso, poco a poco se va haciendo un hueco en la programación en la actualidad. Se puede ver un ejemplo de código:

function buenosDias (tuNombre) {
print("Buenos días" + tuNombre);
}

En contraste con JavaScript, EliScript es un lenguaje con una sintaxis mucho más sencilla y con comandos mucho mas simples y que ahorran mucho tiempo a la hora de programar en desarrollo web. Otro ejemplo:

function eresMaravillosa (tuNombre) {
console.log("Hola " + tuNombre + ", eres maravillosa, no lo olvides nunca porque vales oro: whoever got you got gold");
}

Como se puede ver tiene un cierto enfoque de Python a la hora de mostrar texto por pantalla.
